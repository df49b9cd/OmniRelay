- Root: solution `OmniRelay.slnx`, shared props (`Directory.Build.props`, `Directory.Packages.props`), global.json (.NET 10), repo guides (AGENTS.md, README.md), Dockerfile.ci, .editorconfig.
- src/: production code
  - OmniRelay: core dispatcher, transports, codecs, middleware, peers
  - OmniRelay.Configuration: host/DI binder and config models
  - OmniRelay.Cli: `omnirelay` System.CommandLine tool
  - Codegen: OmniRelay.Codegen.Protobuf (+ Core), Protobuf.Generator (Roslyn incremental), Protobuf console plugin
  - Diagnostics: Abstractions, Runtime, Probes, Telemetry, Logging, Alerting, Documentation subpackages
  - ResourceLease/ShardStore: Sqlite, Postgres, Relational, ObjectStorage; ResourceLeaseReplicator.* (Grpc, Sqlite, ObjectStorage)
- tests/: xUnit suites
  - Core.UnitTests; Dispatcher.UnitTests; Configuration.UnitTests; Cli.UnitTests; Codegen.Tests; CodeGen.IntegrationTests; IntegrationTests; FeatureTests; HyperscaleFeatureTests; YabInterop; TestSupport helpers; testing-todo.md
- docs/: architecture, AOT/diagnostics/reference guides; samples/: runnable demos
- eng/: scripts (run-ci.sh, run-aot-publish.sh, run-hyperscale-smoke.sh); docker/: CI/hyperscale Dockerfiles
- branding/: assets; analyzer-report/build-warnings logs for reference
- Tooling files: .gitignore/.dockerignore, package-lock.json (frontend/tooling), OmniRelay.sln.DotSettings.user.
