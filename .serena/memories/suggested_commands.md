- Shell: zsh on macOS (Darwin); commands below assume zsh syntax.
- Build solution: `dotnet build OmniRelay.slnx`
- Fast unit slice: `dotnet test tests/OmniRelay.Core.UnitTests/OmniRelay.Core.UnitTests.csproj`
- Full CI parity: `./eng/run-ci.sh` (wraps restore/build/tests)
- Hyperscale/interop smoke (Docker): `docker build -f docker/Dockerfile.hyperscale.ci .`
- Native AOT publish: `./eng/run-aot-publish.sh [rid] [Configuration]` (defaults linux-x64 Release)
- CLI dev loop: `dotnet run --project src/OmniRelay.Cli -- --help`
- Restore/build/test inside CI image: `docker build -f Dockerfile.ci .` or `./eng/run-ci.sh --build-arg DOTNET_VERSION=10.0.100`
- Additional xUnit filters: `dotnet test <proj> --filter Category=<name>`
- Useful macOS utilities: `ls`, `rg` (ripgrep), `find`, `pbcopy/pbpaste`, `sed -n`, `open` for files/URLs.
- Repo structure quick glance: `ls src` or `rg --files -g"*.cs" src/OmniRelay` for navigation.
